ISQIP Project


✕


Designed and made an IoT based lighting system as part of Makeathon for EC-ISQIP'18. The system controls the lighting of a large room or office space by detecting the presence of people in that part of the room. The lighting can also be controlled via a web interface using IoT.

The sensor system was controlled by a NodeMCU which send trigger signals to server when people are detected. The server receives data from sensors and send switching signal to relay system. The relay system consisted of a NodeMCU as controller connected to server. There is also a web interface directly in control with switching system where we can switch on/off lights manually.
